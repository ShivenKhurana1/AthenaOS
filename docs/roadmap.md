# Roadmap

## Phase 0 - Research and decisions

- Decide Linux base (Arch, Debian, Ubuntu, Fedora)
- Lock design language and token system
- Define shell feature scope for v0.1

## Phase 1 - Visual system

- Finalize tokens (color, type, motion)
- GTK theme and icon direction
- Shell layout spec (top bar, dock, overview)

## Phase 2 - Shell prototype

- GNOME Shell extension prototype
- Quick settings and launcher mock
- Accessibility and contrast passes

## Phase 3 - Distro tooling

- Build pipeline for ISO images
- Installer integration
- Update channel and release process

## Phase 4 - Alpha

- Alpha ISO with shell theme + core apps
- Feedback loop for performance and UX
